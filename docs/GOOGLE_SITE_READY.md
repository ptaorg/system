# Googleサイト原本の考え方

このリポジトリでは、説明だけではなく、Googleサイトにそのまま貼って使える原本HTMLを用意しています。

## まず使うファイル

`site/google-site-ready.html`

これは、次の内容を1つにまとめたGoogleサイト貼り付け用HTMLです。

- トップページ
- お知らせ
- 活動紹介
- 資料室
- 入会案内
- 重要事項
- FAQ
- お問い合わせ

## 使い方

Googleサイトでページを1つ作り、次の操作で貼り付けます。

```text
挿入
↓
埋め込む
↓
埋め込みコード
↓
site/google-site-ready.html の全文を貼る
↓
挿入
```

## 各PTAが差し替える場所

- ○○学校PTA
- 年度
- 会費
- 支払い方法
- 問い合わせ先メール
- `FORM_URL_HERE`

`FORM_URL_HERE` は、自分のPTAの入会申込GoogleフォームURLに置き換えてください。

## 重要

このサイトは、加入しない方にフォーム提出を求めるものではありません。
入会申込フォームは、PTAへの加入を希望する方だけが提出するものです。

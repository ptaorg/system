# 配布方針

各PTAにGitHubを読ませる運用にはしません。

## 配布するもの

```text
1. 保護者用Googleサイト原本
2. 役員用管理アプリ原本
3. 入会申込フォーム原本
4. はじめに読む説明書
```

## 各PTAに渡す文面

```text
PTAホームページ・入会申込・管理アプリの原本です。
以下をコピーして、自校PTA用に差し替えて使ってください。

① 保護者用Googleサイト原本
② 役員用管理アプリ原本
③ 入会申込フォーム原本

保護者に案内するのは、公開したGoogleサイトURLと入会申込フォームURLだけです。
管理アプリURL、管理スプレッドシートURL、編集用フォームURLは役員以外へ共有しないでください。
```

## GitHubの位置づけ

GitHubは開発・保管用です。利用者向けの入口にはしません。

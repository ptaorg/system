# 重要事項説明テンプレート

以下は、Googleフォームの説明欄に貼り付けるためのたたき台です。実際に使う場合は、各PTAの規約、会費額、活動内容、会計処理、返金方針、退会方法に合わせて修正してください。

---

## PTA入会申込にあたっての重要事項

本PTAは、保護者・教職員が任意に加入する団体です。入会は自動ではありません。以下の内容を確認したうえで、加入を希望する場合は、フォームの「PTAへの加入を申し込みます」を選択してください。

## 1. 任意加入について

PTAへの加入は任意です。加入しないこと、または退会することによって、児童・生徒が学校教育上の不利益を受けることはありません。

PTAに加入しない場合でも、学校が行う教育活動、学校行事、授業、成績評価、生活指導等とは関係しません。

## 2. 活動内容

本PTAは、家庭、学校、地域との連携を図り、児童・生徒の教育環境に資する活動を行います。主な活動は次のとおりです。

```text
例：広報活動、研修活動、地域連携活動、行事協力、環境整備、会員向け連絡等
```

実際の活動内容は、年度の事業計画、総会資料、規約等で確認してください。

## 3. 会費

会費は次のとおりです。

```text
年額または月額：○○円
徴収時期：○月頃
徴収方法：現金／振込／その他
```

会費は、PTAの活動目的に沿って使用します。会計報告、予算、決算は、総会資料等で会員に示します。

## 4. 会員登録の成立時期

このフォームを送信した時点では、申込受付の段階です。会費のお支払いを本PTAが確認した時点で、会員登録が完了します。

入会を取りやめる場合、誤って送信した場合、または内容を訂正したい場合は、PTAまでご連絡ください。

## 5. 個人情報の取扱い

このフォームで取得する情報は、PTA入会申込の受付、会員管理、会費確認、PTAからの連絡、PTA活動の運営に必要な範囲で利用します。

取得する主な情報は次のとおりです。

```text
保護者氏名、児童・生徒氏名、学年・組、メールアドレス、加入意思、同意記録、受付日時、会員登録状況
```

取得した情報は、PTAの会員管理に必要な範囲で管理し、目的外に利用しません。学校が保有する児童・保護者名簿を当然にPTA名簿として利用するものではありません。

## 6. 退会について

PTAは任意加入の団体であり、退会を希望する場合は退会できます。退会を希望する場合は、次の連絡先までご連絡ください。

```text
連絡先：○○PTA
メール：○○@example.com
```

退会時の会費返還の有無や取扱いは、PTA規約、会計処理、実際の支出状況に基づきます。必要に応じて、各PTAで具体的な返金方針を記載してください。

## 7. 問い合わせ先

```text
○○PTA
メール：○○@example.com
```

# Googleフォーム項目例

このページは、Googleフォームを作るときの項目例です。質問文は、`src/Code.gs` の `FIELD_MAP` と一致させてください。

## フォームタイトル

```text
PTA入会申込フォーム
```

## フォーム説明欄

`docs/consent-template.md` の内容を、各PTAの規約、会費額、活動内容、退会方法、個人情報の取扱いに合わせて編集して貼り付けます。

## 質問項目

### 1. 保護者氏名

形式：記述式  
必須：はい

```text
保護者氏名
```

### 2. 児童・生徒氏名

形式：記述式  
必須：各PTAの運用に応じて判断

```text
児童・生徒氏名
```

### 3. 学年・組

形式：記述式  
必須：各PTAの運用に応じて判断

```text
学年・組
```

### 4. メールアドレス

形式：記述式  
必須：はい

```text
メールアドレス
```

### 5. 加入意思の確認

形式：ラジオボタン  
必須：はい

```text
加入意思の確認
```

選択肢：

```text
PTAへの加入を申し込みます
今回は申し込みません
```

この項目を置かない場合、ツールはフォーム送信自体を申込として扱います。任意加入の説明を明確にするには、加入意思の確認欄を置く運用を推奨します。

### 6. 重要事項に同意します

形式：チェックボックス  
必須：はい

```text
重要事項に同意します
```

選択肢：

```text
上記の説明、会費、個人情報の取扱い、退会方法を確認し、同意します
```

## Code.gs 側の対応箇所

フォームの質問文を変更した場合は、`src/Code.gs` の次の部分も同じ文言に変更します。

```javascript
const FIELD_MAP = {
  guardianName: '保護者氏名',
  childName: '児童・生徒氏名',
  gradeClass: '学年・組',
  email: 'メールアドレス',
  joinIntent: '加入意思の確認',
  consent: '重要事項に同意します',
};
```
